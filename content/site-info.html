infoSobre este site

Eu iniciei o desenvolvimento deste site há bastente tempo, apenas para mexer com CSS e testar algumas animações. Ele ficou parado por bastante tempo, apenas com um aviso de "Página em Construção" e links de contato, com este mesmo visual baseado no Material Design e as animações do menu funcionando.

Achei que seria uma boa ideia utilizá-lo como meu currículo, então voltei a mexer nele, preenchendo com informações curriculares.

Não utilizei nenhuma biblioteca/framework de CSS neste site, eu decidir criar do zero uma aparência com base no Material Design.

Foi utilizado o JQuery e alguns de seus plugins (Color e TouchSwipe) para algumas animações do site. A animação do botão de menu (hambúrguer/seta) foi feita por mim em SVG.

A responsividade foi levada em consideração durante o desenvolvimento do CSS, e também foi adicionado o suporte ao gesto de deslizar do canto para abrir o menu, testado em versões recentes de navegadores mobile (Chrome e Firefox para Android).
